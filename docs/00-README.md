# 📚 Libre WebUI Documentation

**The #1 Privacy-First AI Interface** – Built for 100% offline operation, 500+ AI models, and zero data collection.
Designed to outperform traditional web UIs like Open WebUI and LibreChat in privacy, flexibility, and speed.

:::tip New to Libre WebUI?
**[Start with our 5-minute Quick Start Guide →](./QUICK_START)** to get your first AI chat running immediately!
:::

:::info Why Choose Libre WebUI?
🔒 **100% Privacy** - Zero telemetry, your data stays local  
🎯 **500+ AI Models** - Extensive model support across 7 providers  
⚡ **Quick Setup** - 5-minute installation with Docker or local  
🎨 **Modern Interface** - Clean, responsive design optimized for productivity  
🛡️ **Advanced Security** - Comprehensive authentication and role management  
:::

## 🎯 What is Libre WebUI?

:::info Core Philosophy
**100% free and open source** local-first AI interface that runs entirely on your computer. No data leaves your machine unless you choose to connect external services.
:::

<div className="container">
  <div className="row">
    <div className="col col--4">
      <h3>🔒 Privacy & Security</h3>
      <ul>
        <li><strong>No telemetry or tracking</strong> - Your data stays on your hardware</li>
        <li><strong>Complete offline operation</strong> by default</li>
        <li><strong>Optional external connections</strong> when you need them</li>
        <li><strong>Secure authentication</strong> with role-based access control</li>
      </ul>
    </div>
    <div className="col col--4">
      <h3>🤖 AI Integration</h3>
      <ul>
        <li><strong>Complete Ollama integration</strong> - All API endpoints supported</li>
        <li><strong>Multiple AI providers</strong> - OpenAI, Anthropic, Groq, Gemini, Mistral, GitHub Models, OpenRouter</li>
        <li><strong>500+ models</strong> across 7 external providers</li>
        <li><strong>Automatic fallback</strong> to local Ollama when external services fail</li>
      </ul>
    </div>
    <div className="col col--4">
      <h3>✨ Advanced Features</h3>
      <ul>
        <li><strong>Interactive artifacts</strong> - HTML, SVG, code execution</li>
        <li><strong>Document chat (RAG)</strong> - Upload PDFs, TXT, DOCX and chat with content</li>
        <li><strong>Persona development</strong> - AI personalities with memory and learning</li>
        <li><strong>Plugin architecture</strong> - Extensible system for external services</li>
        <li><strong>Modern UI</strong> - Dark/light themes, responsive design, keyboard shortcuts</li>
      </ul>
    </div>
  </div>
</div>
